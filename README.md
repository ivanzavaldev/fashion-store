# Fashion Store - E-Commerce Website

A modern e-commerce web application for a fictional clothing brand. This project is built as a portfolio piece for my application for an **Ausbildung as a Fachinformatiker für Anwendungsentwicklung** in Germany.

## 🚀 Project Overview
The goal of this project is to develop a fully functional, responsive, and visually appealing online clothing store. Through this project, I aim to demonstrate and improve my frontend development skills, while documenting the entire engineering process like a development diary.

## 🛠️ Tech Stack (Planned)
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox, Grid, Responsive Design)
- **JavaScript (Vanilla)** (Cart logic, filtering, dynamic UI)
- **Git & GitHub** (Version control & documentation)

## 📋 Key Features
- [ ] Homepage with featured collections and promotional banners
- [ ] Product catalog with filtering and sorting features (by price, size, category)
- [ ] Product detail pages
- [ ] Interactive shopping cart (add, remove, update quantities)
- [ ] Responsive design optimized for mobile, tablet, and desktop screens
